Updates

Release 2.0.4 (26-May-2016) Changes from version 2.0.2: The new wraper script OPERA-long-read.pl for scaffolding with long reads from third-generation sequencing technologies (PacBio or Oxford Nanopore). Release 2.0.2 (21-Mar-2015) Changes from version 2.0.1: 1. Format of mapping files will be checked before analysing. An error will be reported if the format(e.g. column number) is not correct; 2. In preprocess_reads.pl, users can specify the temporary directory for sorting mapping files (default directory is current directory). Latest and previous releases can be downloaded here.

Installation

Type "make install" in the root directory for OPERA-LG.
